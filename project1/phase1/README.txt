/* Phase1 Readme */

1. 컴파일 방법

- terminal에서 phase1 디렉토리로 이동 후, 'make' 명령어를 입력 해 컴파일 한다.


2. 실행 방법

- myshell의 실행 파일 이름은 myshell 이다. ./myshell을 입력 해 shell 프로그램을 실행한다.

- shell 프로그램이 실행되면, '> ' 문자와 함께 사용자의 입력을 대기하고 있는 상태가 된다.

- 'ls', 'pwd', 'cd', 'ps' 등의 명령어를 입력하면 bash shell 과 동일한 결과를 보여준다.

- 입력한 명령어는 항상 historyLog 에 저장된다. 'history' 명령어를 입력하면 지금까지 입력한 명령어들을 확인할 수 있다.

- '!!', '!#' 명령어 입력 시, historyLog의 특정 명령어를 다시 사용할 수 있다.

- '!!'는 가장 최근에 실행한 명령어를 실행해주고, '!#'은 #에 번호를 입력하면, 해당 번호의 순서에 맞는 명령어를 실행해준다.

- shell 프로그램을 종료시키고 싶으면 'exit' 명령어를 입력하면 된다.
